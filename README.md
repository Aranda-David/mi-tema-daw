## 🚀 Características principales

* **Custom Theme:** Desarrollado sin plantillas prefabricadas.
* **Modo Oscuro/Claro:** Implementado con variables globales de CSS y persistencia.
* **Arquitectura SCSS:** Organización de estilos modular y optimizada.
* **Diseño Responsive:** Totalmente adaptado a dispositivos móviles y tablets.
* **Gestión Dinámica:** Uso intensivo de *Custom Post Types* y *Advanced Custom Fields*.

---

## 🧩 Plugins Esenciales

Para que el tema funcione correctamente, se han integrado los siguientes complementos:

* **Advanced Custom Fields (ACF):** Para la gestión de la Ficha Técnica de los proyectos.
* **Contact Form 7:** Gestión de formularios de contacto personalizados.
* **Query Monitor:** Utilizado durante el desarrollo para optimización de consultas y rendimiento.

---

## 📦 Instalación y Uso

1.  Clona este repositorio en tu carpeta `/wp-content/themes/`.
2.  Importa el archivo de base de datos incluido: `practicas_db.sql` en tu PHPMyAdmin.
3.  Activa el tema desde el panel de administración de WordPress.
